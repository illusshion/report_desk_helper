# Решения по спорным FAQ-сценариям (из логов admin_report_desk.lua)

Применено без отдельного согласования — выбран самый частый ответ админа или каноничная команда ARP Blue.

| Тема | Варианты в логах | Решение |
|------|------------------|---------|
| GPS Тир vs скиллы | «просто стрелять» vs `/gps 7-2` | Локация тира: `/gps 7-2`. Проверка навыков — сценарий «Скиллы /skill» |
| Слёт номера отеля | `00:03` vs `/price` | `Номера отелей слетают около 00:03` (вопрос про слёт, не про поиск) |
| Счёт в банке | «в самом банке» | Текст без команды — **сценарий не добавлен** (1 раз в логах) |
| Доки на авто | «только по рп» | **Сценарий не добавлен** — RP-ответ, не команда |
| Застрял / help | GG, `/id /ad` | **Сценарий не добавлен** — админы чаще отказывают |
| Найти машину vs аренда | `/x` vs `/home /hotel` | Spawn купленной: `/home /hotel`. Аренда на крыше: `/unrent` или `/x` |
| Баланс Advance Credits | `/st` vs `/mn 1` | `/mn 1` (как в логах для «сколько кредитов») |
| Advance Credits help | `/creditshelp` vs `/mn 1` | `/creditshelp` для «как использовать» |
| Инвентарь vs credits | `/i` vs `/creditshelp` | Разделены negative_keywords |
| Прогресс БП /tasks vs квесты | подсказки по квестам | Keywords только про BP/tasks; квесты игры — **не матч** |
| Батл Пасс /bp | ложные «какая команда…» | Keywords только «батл пасс» / `/bp`; negatives для нарко/замов/тюрьмы |
| Магазины /price | «где купить» anything | Убрано голое «где+купить»; negatives для аксов/катаны/заработка |
| Лицензия — где | длинный текст vs `/liclist` | `/liclist` |
| Микрофон | f2/f9/voice mix | `/voice, F2; настройки F9` |
