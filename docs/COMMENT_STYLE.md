# Стиль комментариев Report Desk

Единые правила для `lib/report_desk_*.lua` и entry-файлов.

## File-header

```lua
--[[ Модуль: кратко — за что отвечает файл и с чем связан. ]]
```

## Функции

Одна строка над `function` / `local function` с нетривиальной логикой:

```lua
-- Перехват RPC INITMENU — блокирует серверное SA-Menu в /sp.
function installDeskSpMenuRpcBlock()
```

Не дублировать имя функции. Не комментировать однострочные геттеры.

## Константы

```lua
local OUTBOUND_INTERVAL = 0.55  -- мин. пауза между sp/st/ans (антiflood сервера), сек
```

Указывать единицы: сек, мс, px, строк чата.

## Секции

`-- === Имя секции ===` только в файлах >300 строк (checker, ui, cheats, spectate_stats).

## Язык

Русский, UTF-8. Логику не менять — только комментарии.
